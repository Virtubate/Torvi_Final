# Torvi Landing Page

A modern, beautiful landing page recreation for Torvi - AI.

## Features

- Responsive design
- Modern UI with backdrop blur effects
- Beautiful hero section with mountain landscape background
- Company logo showcase
- Smooth animations and hover effects

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Libre Baskerville, Inter)

## Setup

1. Simply open `index.html` in your web browser
2. No build process or dependencies required

## Fonts

- **Libre Baskerville** - Used for the main heading (serif font)
- **Inter** - Used for body text, buttons, and navigation
